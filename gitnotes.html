Git Setup

Git Clone - Take a repository from the internat (GitHUb) and download it onto my own computer.
Git config global user (email and name) - sets up the link between my computer and GitHUb you need to do both email and name commands.
Git remote add origin - makes the link to the repository from local to online, you need the URL from GitHub

Using Git

Git Add - tells git that I want to add a file and track it (eg: git add hello.html) it doesn't make the commit, just lets git know.
Git Commit - tells git to make the copy online and keep track of changes, the messsgae helps you remember the version. You add a comment.
Note: Both of these changes occur locally. If you want to see them online, you must push the changes to the GitHub
Git Push - sends all the changes to the online GitHub, now you can see the files online and even open them
Git Status - tells me what is currently going on with my git. It says which location, branch and how it is all working
Git Commit am - combines the add and commit statements together and tells git to add all the files that have been changed.
Git Pull - gets the latest version from GitHUb and puts it on my local machine
Git log - kep track of all the changes you have made over time
